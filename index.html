Projects

  • Stonade - A marketplace web app

    Description: A highly complex and extensive app and admin panel for a marketplace website. This app was developed for selling construction and decorative stones in countries such as the UK and Turkey. The admin panel includes numerous features for sales, purchases, transfers, cost calculations, returns, inventory management, and much more, making it a large-scale system. It also provides a comprehensive analytics section with options to export data, statistics, and invoices in PDF and Excel formats. The project is built using NestJS, Prisma, Event-driven architecture, and AWS. Email sending and receiving are implemented through external services. The deployment is done in Turkey using Docker Compose on a VFS with Nginx. Overall, this project has provided extensive experience and knowledge. Additionally, comprehensive test coverage has been implemented using Jest.

    Technologies Used: JavaScript, Node.js, Nestjs, Postgres, Prisma, AWS, Docker, VFS, Nginx, Firebase

    Date: July 2024 - April 2025

  • Airshow - Iran airshow in Kish island

    Description: An app and admin panel for the Iran Airshow Exhibition on Kish Island. The admin panel includes various features such as managing participants and exhibitors, handling different registration and payment stages, issuing exhibitor invoices, and sending emails and SMS notifications. Additionally, it supports generating PDFs for invoices, embassy approval requests, and more, as well as exporting data in Excel format. The project is built using PostgreSQL, Prisma, NestJS, and other technologies.

    Technologies Used: JavaScript, Node.js, Postgres, Prisma, Docker

    Date: May 2024 - July 2024
